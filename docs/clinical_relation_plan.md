# 面向临床病历的 CMeKG 关系体系扩展方案

> 基于两份医疗数据集分析：
> - **CMB-test-qa.json**（~70 个外科临床病例分析，考试型描述）
> - **subtask1_test_set_with_answer.json**（379 条肿瘤科真实 EMR 病程记录，医渡云标注）
>
> 版本：v4 — 完成 CMB 临床病例高频关系补全，最终谓词体系 59 个

## 现状分析

### CMeKG 现有 23 种关系谓词

| 类别 | 谓词 | 数量 |
|------|------|:----:|
| **药品说明相关** | 用法用量、贮藏、性状、不良反应、有效期、注意事项、适应症、成份、规格、药物相互作用、主治、入药部位、功效、性味、功能主治 | **15** |
| **疾病-症状-诊疗** | 相关疾病、相关症状、临床表现、病因、并发症、治疗 | **6** |

## 数据集分析

### CMB 临床病历（外科考试病例）

- **70 条**，结构化描述（现病史→体格检查→辅助检查→诊断→治疗）
- 实体覆盖：疾病、症状、解剖部位、检查、药物、手术
- 核心关系：症状-部位、检查-发现、体征-数值、诊断鉴别、治疗
- 独特贡献：体格检查体征（腹软、压痛）、否定表达、鉴别诊断、相对位置

### Yidu 肿瘤 EMR（真实电子病程记录）

- **379 条**，平均 350 字/条，平均标注 **15 个实体**/条
- 6 种预标注实体类型分布：解剖部位 **3094** > 疾病和诊断 **1323** > 实验室检验 **590** > 药物 **485** > 影像检查 **348** > 手术 **162**
- 独特贡献：化疗方案、肿瘤分期、免疫组化、疗效评价、基因检测、药物剂量

### 关键重叠分析（详细）

#### 1. 相对位置 #52 ↔ 病变部位 #25 → **建议合并**

```
病变部位 #25: (肿块, 病变部位, 右侧腹股沟区)     ← 一般位置
相对位置 #52: (肿块, 相对位置, 腹股沟韧带上内方)  ← 精确位置
```

两者语义本质相同，只是精度不同。标注时难以界定"何时用病变部位何时用相对位置"。「腹股沟韧带上内方」本身就是一个位置描述，完全可以用 #25 病变部位覆盖。

**结论：删除 #52，统一用 #25 病变部位**

#### 2. 对比评估 #55 ↔ 疗效评价 #45 → **建议合并**

```
疗效评价 #45: (FOLFOX, 疗效评价, SD)          ← RECIST标准化评级
对比评估 #55: (病灶, 对比评估, 较前无明显变化)   ← 影像描述
```

在肿瘤学中，"疗效评价=SD"的判断依据正是"较前无明显变化"。两者指向相同临床概念，只是在表述方式上一个是标准化缩写、一个是自然语言描述。合并后语义更完整。

**结论：删除 #55，将"对比评估"纳入 #45 疗效评价的值域**

#### 3. 药物剂量 #49 ↔ 用法用量 #5 → **需保留，但明确边界**

```
用法用量 #5（原有）: (阿莫西林, 用法用量, 口服 一次0.5g 一日3次)  ← 完整用法说明
药物剂量 #49（新增）: (泰素, 药物剂量, 240MG)                       ← 仅数值+单位
```

临床病历中几乎不出现完整"用法用量"句式，而以"泰素240MG"为主。两者粒度不同。

**结论：保留 #49，标注时约定"用法用量用于完整用法，药物剂量仅用于数值+单位"**

#### 4. 体征 #50 ↔ 临床症状/临床表现 → **需保留，但明确边界**

```
临床表现（原有）: (胃食管反流病, 临床表现, 反酸/烧心)          ← 疾病→症候群
相关症状（原有）: (糖尿病, 相关症状, 多饮/多尿)                ← 疾病→症状列表
体征 #50（新增）: (腹, 体征, 软) / (肿块, 体征, 压痛)          ← 部位→体检发现
```

体征是医生体检时的触诊/叩诊/听诊发现，主语是解剖部位。症状是患者主诉。主语不同、发现方式不同。

**结论：保留 #50**

#### 5. 考虑诊断 #57 ↔ 诊断 #40 → **需保留**

```
诊断 #40: (患者, 诊断, 嵌顿性腹股沟斜疝)          ← 确诊
考虑诊断 #57: (患者, 考虑诊断, 卵巢来源肿瘤)       ← 待排/初步诊断
```

临床文本中"考虑""可能""待查"与确诊严格区分。合并会丢失这一关键信息。

**结论：保留 #57**
